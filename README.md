# taschenrechner

1. Build process:

cmd: make
-> generiert main und taschenrechner

2. Eingabemethode:

a ... addition

s ... subtraktion

m ... multiplikation

d ... division

pow ... potenzieren

root_of ... wurzel von ..

r ... Modulu

Beispiel einer Eingabe:

cmd: main \n

cmd: 23 a 7 m 2 r 2

Info: Die eingabe von Parameter ist auf 100 Zeichen begrenzt
